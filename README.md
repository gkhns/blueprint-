#Windows # #



**NMAP Scan**

```sh
  nmap -sV -sC -p- 10.10.3.65
  ```

- To see the versions of the services running (**-sV**)
- To perform a script scan using the default set of scripts (**-sC**)
- To scan all ports from 1 through 65535 (**-p-**)
